# AppliedDataScienceCapstone

Applied data science capstone project that gives the chance to work with data science methodology involving data collection, data wrangling, exploratory data analysis, data visualization, model development and  evaluation, and reporting results to stakeholders.

The developer has no affilation to the data used in the capstone project as the data is used as learning dataset to go through  data science Methedology for model development and evaluation in data science project.
